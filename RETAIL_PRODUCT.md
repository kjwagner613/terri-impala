# ImpalaStreamer — Packaged Retail Product

This document describes the packaged retail version of ImpalaStreamer.

## What’s Included
The retail version provides a complete, sovereign media system:

- core player  
- local library  
- cloud library  
- webcast module  
- evolving cultivated catalog  
- DD microservices  
- serial number ownership  
- DD registry lookup  
- updates and support  

## Why It’s Different From the Repo
The open-repo version is a bare-metal core intended for developers.

The retail version is:

- plug-and-play  
- polished  
- supported  
- sovereign  
- feature-complete  
- ready for everyday users  

It includes components not available in the repo, including microservices, catalog systems, and the ownership/serial infrastructure.

## Serial Number System
Each retail copy includes a unique serial number:

- embedded into the build  
- tied to the owner  
- permanent  
- recorded in the DD registry  
- used for provenance and verification  

## Pricing
Expected retail price: **$30–$50 USD**.

## Ownership
Retail customers own their copy.  
Pilot/evaluation users do not — their serials remain property of Discrete Development LLC.
