# ImpalaStreamer — Commercial Licensing

This document explains commercial licensing for the packaged retail version of ImpalaStreamer.

## Commercial Use Requires a License
Any use of ImpalaStreamer by a business, organization, or for-profit entity requires a paid commercial license.

Commercial use includes:

- use within a company  
- use in paid services  
- use in monetized workflows  
- use in production environments  
- use as part of a commercial offering  
- use of the packaged retail version  

## Packaged Retail Product
The packaged version includes:

- sovereign core player  
- local library  
- cloud library  
- webcast module  
- cultivated catalog  
- DD microservices  
- updates and support  
- a unique serial number representing ownership  

Expected retail price: **$30–$50 USD**.

## Serial Number Ownership
Each purchased copy receives a unique serial number:

- embedded into the build  
- permanent  
- non-transferable without permission  
- recorded in the DD registry  

Serial Number **#1** is reserved for the creator, Kevin Wagner.
